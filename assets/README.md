## Download Instructions

### Auto-downloaded (via script)
**Asset #1** — `scripts/download_assets.ps1` will download and extract this automatically.

### Manual download required
Assets **#2–#5** are hosted on itch.io which does not provide stable direct-download URLs.
Run `scripts/download_assets.ps1` and follow the printed instructions, or manually:

1. Open the asset's **Source URL** in your browser
2. Click the download / "Download Now" button
3. Extract the ZIP contents into the **Extract To** directory listed above
4. Delete the ZIP after extraction (optional)
